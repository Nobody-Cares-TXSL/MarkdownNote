# VUE
## npm
### 配置npm
```bash
# 配置npm镜像源为阿里云镜像源
npm config set registry https://registry.npmmirror.com

# 查看镜像源
npm config get registry
```

### 命令
```bash
# 初始化项目，生成 package.json 文件
npm init

# 安装指定的包
npm install <package-name>

# 卸载指定的包
npm uninstall <package-name>

# 运行 package.json 中定义的脚本命令
npm run <script-name>
```
