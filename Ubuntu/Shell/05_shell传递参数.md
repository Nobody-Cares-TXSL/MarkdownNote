# Shell 传递参数

## 基本用法

脚本内通过 `$n` 获取参数，n 为数字。
> 例如：`$1` 获取第一个参数，`$2` 获取第二个参数，以此类推。
>
> `$0` 获取脚本本身的名称。

## 特殊参数变量

| 变量 | 说明 |
|------|------|
| `$#` | 参数个数 |
| `$*` | 所有参数（作为**一个**字符串） |
| `$@` | 所有参数（每个参数**独立**字符串） |
| `$$` | 当前进程 PID |
| `$!` | 后台最后一个进程 PID |
| `$-` | Shell 当前选项 |
| `$?` | 上条命令退出状态（0=成功） |

## `$*` vs `$@` 核心区别

在**双引号**中才有区别（假设参数为 1 2 3）：

- `"$*"` → `"1 2 3"`（一个整体）
- `"$@"` → `"1" "2" "3"`（三个独立参数）

```bash
for i in "$*"; do echo $i; done   # 输出: 1 2 3（一行）
for i in "$@"; do echo $i; done   # 输出: 1\n2\n3（三行）
```

## 注意事项

- 参数含空格时，传递时必须用引号括起：`./test.sh "a b" "c d"`
- 函数传递参数时也必须加引号：`my_func "$@"`
- 校验参数是否存在：`if [ -n "$1" ]; then ... fi`
