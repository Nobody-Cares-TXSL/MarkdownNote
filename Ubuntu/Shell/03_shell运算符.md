# Shell 运算符

> 原文：[Shell 基本运算符 | 菜鸟教程](https://www.runoob.com/linux/linux-shell-basic-operators.html)

## 核心要点

- 原生 bash 不支持数学运算，需借助 `expr`、`awk` 等命令
- 表达式和运算符之间**必须有空格**：`2 + 2`（不是 `2+2`）
- 完整表达式用**反引号** `` ` `` 包裹（Esc 键下方）
- 乘号 `*` 前必须加反斜杠转义：`\*`
- 条件表达式放在 `[ ]` 中，方括号内侧**必须有空格**

## 算术运算符

| 运算符 | 说明 | 示例（a=10, b=20） |
|--------|------|---------------------|
| `+` | 加法 | `` `expr $a + $b` `` → 30 |
| `-` | 减法 | `` `expr $a - $b` `` → -10 |
| `*` | 乘法 | `` `expr $a \* $b` `` → 200 |
| `/` | 除法 | `` `expr $b / $a` `` → 2 |
| `%` | 取余 | `` `expr $b % $a` `` → 0 |
| `=` | 赋值 | `a=$b` |
| `==` | 相等比较 | `[ $a == $b ]` → false |
| `!=` | 不等比较 | `[ $a != $b ]` → true |

> MAC 中使用 `$((表达式))` 语法，`*` 无需转义。

## 关系运算符（仅支持数字）

| 运算符 | 说明 | 示例（a=10, b=20） |
|--------|------|---------------------|
| `-eq` | 等于 | `[ $a -eq $b ]` → false |
| `-ne` | 不等于 | `[ $a -ne $b ]` → true |
| `-gt` | 大于 | `[ $a -gt $b ]` → false |
| `-lt` | 小于 | `[ $a -lt $b ]` → true |
| `-ge` | 大于等于 | `[ $a -ge $b ]` → false |
| `-le` | 小于等于 | `[ $a -le $b ]` → true |

## 布尔运算符

| 运算符 | 说明 | 示例（a=10, b=20） |
|--------|------|---------------------|
| `!` | 非运算 | `[ ! false ]` → true |
| `-o` | 或运算 | `[ $a -lt 20 -o $b -gt 100 ]` → true |
| `-a` | 与运算 | `[ $a -lt 20 -a $b -gt 100 ]` → false |

## 逻辑运算符（使用 `[[ ]]`）

| 运算符 | 说明 | 示例（a=10, b=20） |
|--------|------|---------------------|
| `&&` | 逻辑 AND | `[[ $a -lt 100 && $b -gt 100 ]]` → false |
| `\|\|` | 逻辑 OR | `[[ $a -lt 100 \|\| $b -gt 100 ]]` → true |

> **注意**：`&&` 和 `||` 必须在双括号 `[[ ]]` 中使用，`-a` / `-o` 在单括号 `[ ]` 中使用。

## 字符串运算符

| 运算符 | 说明 | 示例（a="abc", b="efg"） |
|--------|------|--------------------------|
| `=` | 相等 | `[ $a = $b ]` → false |
| `!=` | 不等 | `[ $a != $b ]` → true |
| `-z` | 长度为0 | `[ -z $a ]` → false |
| `-n` | 长度不为0 | `[ -n "$a" ]` → true |
| `$` | 非空 | `[ $a ]` → true |

## 文件测试运算符

| 运算符 | 说明 |
|--------|------|
| `-e` | 文件是否存在 |
| `-f` | 是否为普通文件（非目录/设备文件） |
| `-d` | 是否为目录 |
| `-r` | 是否可读 |
| `-w` | 是否可写 |
| `-x` | 是否可执行 |
| `-s` | 文件是否不为空（大小 > 0） |

```bash
# 示例
if [ -f /var/log/syslog ]; then
    echo "文件存在且为普通文件"
fi
```

## 速记口诀

- **算术**：`+ - \* / %` 用 `expr`
- **关系**：`-eq -ne -gt -lt -ge -le`（equal/not/greater/less/greater-equal/less-equal）
- **布尔**：`! -o -a` 在 `[ ]` 中
- **逻辑**：`&& ||` 在 `[[ ]]` 中
- **字符串**：`= != -z -n`
- **文件**：`-e -f -d -r -w -x -s`
