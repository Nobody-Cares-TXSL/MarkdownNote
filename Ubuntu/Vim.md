# Vim 

## 安装 Vim
```bash
sudo apt update && sudo apt upgrade -y
sudo apt install vim -y
```

## Vim 基础使用

### Vim 的四种模式

| 模式 | 进入方式 | 用途 |
|------|---------|------|
| **普通模式 (Normal)** | 打开文件默认进入 | 浏览、复制、粘贴、删除 |
| **插入模式 (Insert)** | 按 `i`, `a`, `o` 等 | 编辑文本内容 |
| **命令模式 (Command)** | 按 `:` (冒号) | 保存、退出、搜索替换 |
| **可视模式 (Visual)** | 按 `v`, `V`, `Ctrl+v` | 选择文本块 |

### 快速入门

#### 打开/保存/退出
```vim
:q          # 退出（未修改时）
:q!         # 强制退出（放弃修改）
:w          # 保存
:wq         # 保存并退出
:x          # 同 :wq
```

#### 光标移动（普通模式）
```vim
h j k l     # 左 下 上 右
w / b       # 下一个/上一个单词开头
0 / $       # 行首/行尾
gg / G      # 文件首/文件尾
:数字       # 跳到指定行，如 :10
```

#### 进入插入模式
```vim
i           # 在光标前插入
a           # 在光标后插入
o           # 在下方新建一行插入
I           # 在行首插入
A           # 在行尾插入
O           # 在上方新建一行插入
Esc         # 返回普通模式
```

#### 编辑操作
```vim
x           # 删除光标处字符
dd          # 删除当前行
dw          # 删除到单词末尾
d$          # 删除到行尾
yy / Y      # 复制当前行
p           # 在光标后粘贴
P           # 在光标前粘贴
u           # 撤销
Ctrl+r      # 重做
```

#### 搜索替换
```vim
/关键词      # 向下搜索
?关键词      # 向上搜索
n / N       # 下一个/上一个搜索结果
:%s/旧/新/g  # 全文替换
:%s/旧/新/gc # 全文替换（需确认）
```

#### 可视模式操作
```vim
v           # 字符可视模式
V           # 行可视模式
Ctrl+v      # 块可视模式
d           # 删除选中内容
y           # 复制选中内容
```
