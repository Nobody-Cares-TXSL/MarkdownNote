# 正则表达式（Regex）

> 正则表达式是一组由字母和符号组成的特殊文本，用于从文本中找出满足格式的字符串。常用于**搜索替换**、**表单验证**、**字符串提取**。

---

## 1. 元字符

| 元字符 | 描述 | 示例 |
|:------:|------|------|
| `.` | 匹配除换行符外的任意单个字符 | `.ar` → car, par, gar |
| `[ ]` | 字符集，匹配方括号内任意字符 | `[Tt]he` → The, the |
| `[^ ]` | 否定字符集，匹配方括号外任意字符 | `[^c]ar` → par, gar（不匹配 car） |
| `*` | 前一个字符出现 **>=0** 次 | `a*` → "", a, aa, aaa |
| `+` | 前一个字符出现 **>=1** 次 | `c.+t` → cat, c123t |
| `?` | 前一个字符出现 **0 或 1** 次 | `[T]?he` → he, The |
| `{n,m}` | 前一个字符出现 **n 到 m** 次 | `[0-9]{2,3}` → 12, 345 |
| `{n,}` | 前一个字符出现 **>=n** 次 | `[0-9]{2,}` → 12, 34567 |
| `{n}` | 前一个字符出现 **恰好 n** 次 | `[0-9]{3}` → 123 |
| `(...)` | 捕获组，视为整体 | `(ab)*` → "", ab, abab |
| `(?:...)` | 非捕获组 — 匹配内容但不存入分组变量 | `(?:c\|g\|p)ar` 匹配 car/gar/par，但 `$1` 等分组变量中无值 |
| `\|` | 或运算符 | `(T\|t)he\|car` → The, the, car |
| `\\` | 转义特殊字符 | `\\.` 匹配字面 `.` |
| `^` | 匹配行首 | `^(T\|t)he` → 仅行首的 The/the |
| `$` | 匹配行尾 | `(at\\.)$` → 仅行尾的 at. |

---

## 2. 简写字符集

| 简写 | 等价 | 描述 |
|:----:|------|------|
| `.` | `[^\n\r]` | 除换行符外的所有字符 |
| `\w` | `[a-zA-Z0-9_]` | 字母、数字、下划线 |
| `\W` | `[^\w]` | 非字母数字 |
| `\d` | `[0-9]` | 数字 |
| `\D` | `[^\d]` | 非数字 |
| `\s` | `[\t\n\f\r\p{Z}]` | 空白字符 |
| `\S` | `[^\s]` | 非空白字符 |
| `\n` | — | 换行符 |
| `\r` | — | 回车符 |
| `\t` | — | 制表符 |
| `\f` | — | 换页符 |
| `\v` | — | 垂直制表符 |
| `\R` | `\r\n` / `\r` / `\n` | 任意换行符（匹配 CR、LF 或 CRLF） |

## 3. 零宽度断言（前后预查）

> 非捕获组，用于匹配模式的前后条件，不消耗字符。

| 符号 | 描述 | 示例 |
|:----:|------|------|
| `(?=...)` | 正先行断言 — 后面跟着 `...` 所定义的表达式 | `(T\|t)he(?=\sfat)` → The fat 中的 The |
| `(?!...)` | 负先行断言 — 后面不跟 `...` 所定义的表达式 | `(T\|t)he(?!\sfat)` → the mat 中的 the |
| `(?<=...)` | 正后发断言 — 前面跟着 `...` 所定义的表达式 | `(?<=\$)[0-9\.]*` → `$4.44 和 $10.88` 中的 4.44, 10.88 |
| `(?<!...)` | 负后发断言 — 前面不跟 `...` 所定义的表达式 | `(?<!(T\|t)he\s)cat` → alone cat 中的 cat |

---

## 4. 标志（修饰符）

> 示例中的 `/` 是 JavaScript 正则字面量的定界符，非正则语法本身。Python 等语言通过 `re.IGNORECASE` 等参数传入标志。

| 标志 | 描述 | 示例 |
|:----:|------|------|
| `i` | 忽略大小写 | `/The/gi` → 匹配 The, the, THE |
| `g` | 全局搜索（返回全部匹配） | `/.(at)/g` → fat, cat, sat, mat |
| `m` | 多行模式（`^` `$` 匹配每行） | `/at(.)?$/gm` → 每行末尾的 at |

---

## 5. 贪婪与惰性

| 模式 | 描述 | 示例 |
|:----:|------|------|
| 贪婪（默认） | 匹配尽可能长的子串 | `/(.*at)/` → **The fat cat sat on the mat** |
| 惰性 `?` | 匹配尽可能短的子串 | `/(.*?at)/` → **The fat** |

> 在量词后加 `?` 切换为惰性模式：`*?` `+?` `??` `{n,m}?`
